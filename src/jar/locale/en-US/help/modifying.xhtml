Modifying Headers

If you wish to change the value of a header to a new value, discarding the previous value, use the Modify option. This is suitable for headers that can only have a single value, eg:

  • Host
  • User-Agent

Note that if this option is used to specify a header that does not exist by default, a header with the selected name will be created.
