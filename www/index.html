Current release: v0.5.0, 4 December 2005

The fourth public release of Modify Headers adds support for Firefox 1.5, a new option for running the extension while the window (or tab) is closed and a more detailed help function.

Previous release: v0.3.0, 4 May 2005

The third release of the Modify Headers extension for Mozilla Firefox is now available.

The main feature added for this release is help documentation to provide uses with guidance in using the extension. In addition, Enable All and Disable All buttons have been added.

modifyheaders Overview

This extensions allows the user to Add, Modify and Filter out HTTP request headers. See the help tab in the Modify Headers window for more information.
