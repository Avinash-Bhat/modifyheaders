Modify Headers Overview

This extensions allows the user to Add, Modify and Filter out HTTP request headers. See the help tab in the Modify Headers window for more information.

TIP: Use this with XHTML Mobile Profile and the WML Browser for testing mobile sites.

Releases

Development Release: v0.6.1, 20 May 2007

This release has an experimental UI:

  • The tabs have been removed.
  • The Help pages are now hosted externally.
  • The configuration screen pops up from the bottom of the page.

If you download the new version, please take the time to provide feedback on the modifyheaders mailing list. As always please report bugs at http://modifyheaders.mozdev.org/bugs.html

Current Release: v0.6.0, 29 April 2007

This release adds an import/export feature. Access the wizard on the configuration tab. Install at Mozilla Update

Release for Nvu: v0.5.2-nvu, 10 September 2006

Modify Headers has been ported to the Nvu editor platform. Go ahead and try it out (Firefox/Flock/Mozilla users, right click and 'Save Link As...').
